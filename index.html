Getting Connected to the ILIA Intelligence Hub

Follow these steps to get your Claude account fully connected to the NIW ILIA platform

⏳ Hub in active build — full onboarding available when ready
1
Create Your Anthropic / Claude Account
Sign up for Claude at claude.ai using your NIW work email address. If your organization moves to Claude Team or Enterprise, you'll receive an invite — use that instead.
→ Go to claude.ai Account Settings
2
Install Claude Desktop (Optional but Recommended)
Download Claude for Mac or Windows for the best experience. Works alongside the web version — same account, same projects.
→ Download Claude
3
Install Claude in Chrome Extension
Enables Claude to see and interact with your browser — navigate pages, click buttons, and assist with web-based workflows in ILIA and other tools.
→ Chrome Web Store
4
Enable Required Connectors
In Claude Settings → Connectors, enable each of the following. These allow Claude to read and act across NIW tools.
📧 Gmail 📁 Google Drive 📅 Google Calendar 📌 JIRA (Atlassian) 📘 Confluence ⚙️ GitHub
5
Join the ILIA Central Intelligence Claude Project
Once the NIW Claude Team plan is active, you'll receive an invite to the shared ILIA CIH Project. This gives your Claude account full ILIA platform context — workflows, architecture, SOPs, and agent configurations. Until then, Jamin will share the project instruction packet with you directly.
○ Pending Team Plan
6
Select Your Agent Role
Once in the project, select the agent configuration for your role — Operations, Marketing, Customer Service, Dev Assistant, or Onboarding. Each is pre-configured with the right context, tone, and tool access for your function.
○ See Agent Directory below
